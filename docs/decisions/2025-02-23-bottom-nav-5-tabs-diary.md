# 하단 네비게이션 5탭 확장 및 다이어리 권한별 분기

**날짜:** 2025-02-23

## 결정 내용
- 기존 3탭(홈/사연등록/마이페이지) → 5탭(홈/탐색/작성/투데이/마이페이지)으로 확장
- 작성(Diary) 탭: 유저 권한(환자/후원자/비로그인)별 화면 분기

## 이유
- 기획안에 따른 기능 재구성: 탐색·투데이·다이어리 독립 탭 필요
- 환자: 투병 다이어리·감사편지·내 게시물 관리
- 후원자: 내가 후원 중인 환자 목록 → 투병/감사편지 조회
- 비로그인: 로그인 유도 바텀시트로 자연스러운 유도

## 기술적 선택
- **IndexedStack**: 5탭 children 모두 유지하여 스크롤/상태 보존
- **아웃라인 아이콘**: Icons.*_outlined 사용으로 선명한 시각
- **LoginPromptDialog.showAsBottomSheet**: 작성 탭 비로그인 시 바텀시트로 노출
- **관리자 5탭**: 별도 유지 (홈/컨트롤타워/추가/마이페이지/관리자설정)
