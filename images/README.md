# WITH 플랫폼 이미지 에셋 (루트 images/)

- **실제 배포용 경로**: `assets/images/` 를 사용합니다. 이미지 파일은 **`assets/images/`** 폴더에 두고, 코드·pubspec은 `assets/images/파일명`으로 참조합니다.
- 이 폴더(`images/`)에 파일이 있다면 `assets/images/`로 복사한 뒤 대소문자·확장자가 코드와 일치하는지 확인하세요.
- **파일명**: 공백 없이 언더바, 대소문자 정확히 (예: `mascot_yellow.png`, `mascot_pink.png`).
