# WITH 플랫폼 이미지 에셋

- **위치**: 프로젝트 루트 `images/`. (Flutter 웹 빌드 시 `assets/assets/` 중복을 피하기 위해 `assets/images/` 대신 사용)
- **파일명**: 공백 없이 **언더바(_)** 사용 (예: `mascot_p.png`).
- **필요한 이미지**:  
  `mascot_yellow.png`(스플래시·로딩·메인 카드), `mascot_pink.png`, `mascot_green.png`, `mascot1.jpg`, `mascot1.png`, `mascot3.png`, `mascots.jpg`
